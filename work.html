To-Do List App

The To-Do List App is a simple web application built using HTML, CSS, and jQuery that allows users to create and manage a list of tasks. Users can add new items to the list by typing in the input field, cross out completed items by double-clicking on them, and delete items by clicking on the "X" button. The app also includes a functionality for reordering the items on the list.
